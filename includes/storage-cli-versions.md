## <a name="versions-of-the-azure-cli"></a>Azure CLI のバージョン

現在使用可能な Azure コマンド ライン インターフェイス (CLI) のバージョンは 2 つあります。

* [Azure CLI](../articles/storage/common/storage-azure-cli.md):Azure CLI の現在のバージョン。Resource Manager デプロイ モデルで使用します。
* [Azure クラシック CLI](../articles/storage/common/storage-azure-cli-nodejs.md):Azure CLI の旧バージョン。従来のデプロイ モデルでのみ使用できます。

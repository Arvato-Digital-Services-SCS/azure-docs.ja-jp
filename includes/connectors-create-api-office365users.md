## <a name="prerequisites"></a>前提条件

* [Office 365 Users](https://office365.com) アカウント  

ロジック アプリで Office 365 Users アカウントを使用するには、Office 365 Users アカウントに接続するロジック アプリを承認しておく必要があります。これは、Azure Portal のロジック アプリ内から簡単に実行できます。  

Office 365 Users アカウントに接続するロジック アプリを承認する手順を次に示します。  

1. Office 365 Users への接続を作成するには、ロジック アプリ デザイナーのドロップダウン リストから **[Microsoft が管理している API を表示]** を選択し、検索ボックスに「*Office 365 Users*」と入力します。 使用するトリガーまたはアクションを選択します。  
   ![Office 365 Users の接続の作成手順](./media/connectors-create-api-office365users/office365users-1.png)  
2. これまでに Office 365 Users への接続を作成したことがない場合は、Office 365 Users の資格情報の入力を求められます。 この資格情報を使用して、接続するロジック アプリの承認と、Office 365 Users アカウントのデータへのアクセスが行われます。  
   ![Office 365 Users の接続の作成手順](./media/connectors-create-api-office365users/office365users-2.png)  
3. Office 365 Users のユーザー名とパスワードを入力して、ロジック アプリを承認します。  
   ![Office 365 Users の接続の作成手順](./media/connectors-create-api-office365users/office365users-3.png)  
4. 接続が作成されたら、ロジック アプリで他の手順を自由に実行できるようになります。  
   ![Office 365 Users の接続の作成手順](./media/connectors-create-api-office365users/office365users-4.png)

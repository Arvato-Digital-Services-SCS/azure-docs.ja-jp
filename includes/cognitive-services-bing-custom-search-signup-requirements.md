## <a name="create-an-azure-resource"></a>Azure リソースを作成する

次のいずれかの Azure リソースを作成して、Bing Custom Search API の使用を開始します。

### <a name="trial-resource"></a>[試用版のリソース](https://azure.microsoft.com/try/cognitive-services/?api=bing-custom-search-api)
   * Azure サブスクリプションは不要です。
   * 7 日間有効です (無料)。 試用版のキーとエンドポイントは、サインアップ後に [Azure Web サイト](https://azure.microsoft.com/try/cognitive-services/my-apis/)で入手できます。

### <a name="bing-custom-search-resource"></a>[Bing Custom Search リソース](https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesBingCustomSearch)
   * ご自身でリソースを削除するまでは Azure portal からご利用いただけます。
   * Free 価格レベルを使ってサービスを試用し、後から運用環境用の有料レベルにアップグレードします。

### <a name="multi-service-resource"></a>[マルチサービス リソース](https://ms.portal.azure.com/#create/Microsoft.CognitiveServicesAllInOne)
   * ご自身でリソースを削除するまでは Azure portal からご利用いただけます。  
   * 複数の Cognitive Services 全体で同じキーとエンドポイントをアプリケーションに使用します。

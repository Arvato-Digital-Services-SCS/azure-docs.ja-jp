次の制限が Azure Site Recovery に適用されます。

| 制限の種類 | 制限 |
| --- | --- |
| サブスクリプションあたりの資格情報コンテナーの数 |500 |     
| Azure 資格情報コンテナーあたりのサーバーの数 |250 |
| Azure 資格情報コンテナーあたりの保護グループの数 |制限なし |
| Azure 資格情報コンテナーあたりの復旧計画の数 |制限なし |
| 保護グループあたりのサーバーの数 |制限なし |
| 復旧計画あたりのサーバーの数 |50 |

> [!NOTE]
> Azure Container Service の Docker Swarm オーケストレーターは、従来のスタンドアロンの Swarm を使用します。 現時点では、(Docker 1.12 以降の) 統合された [Swarm モード](https://docs.docker.com/engine/swarm/)は、Azure Container Service でサポートされているオーケストレーターではありません。 Azure で Swarm モードのクラスターをデプロイする場合は、オープンソースの [ACS Engine](https://github.com/Azure/acs-engine/blob/master/docs/swarmmode.md)、コミュニティ提供の[クイックスタート テンプレート](https://azure.microsoft.com/resources/templates/101-acsengine-swarmmode/)、または [Azure Marketplace](https://azuremarketplace.microsoft.com) の Docker ソリューションを使用してください。

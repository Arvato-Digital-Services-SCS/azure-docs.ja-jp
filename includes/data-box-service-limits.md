 - Data Box サービスで複数のストレージ アカウントを使用している場合、すべてのストレージ アカウントが同じ Azure リージョンに属している必要があります。
 - 3 つの以内のストレージ アカウントを使用することをお勧めします。 それ以上のストレージ アカウントを使用するとパフォーマンスに影響する可能性があります。

# <a name="co-sell-partner-engagement"></a>共同販売のパートナー エンゲージメント

Microsoft の共同販売プログラムを使用すると、パートナーは Microsoft と直接連携して共同販売を行うことができます。 膨大なコミュニティのお客様を対象にして、ビジネスの成長を加速させる新しい営業案件を作り出すことができます。

## <a name="partner-selling-motions"></a>パートナーの販売の動き

パートナーの販売には、次の 3 つの動きがあります。

* *パートナーによる販売*:独自の販売元を介して Microsoft テクノロジを再販できます。
* *サービスの共同販売*:販売元とパートナーは協力して、Microsoft テクノロジが埋め込まれたプロジェクトまたはマネージド サービスを販売します。
* *IP 共同販売*:販売元/パートナーは協力して、反復可能でパッケージ化された IP ソリューションを販売します*。

\* こうした IP 共同販売ソリューションの一部は、Microsoft のフィールド セールス チームと共に販売および展開に成功した場合に、追加のインセンティブを受け取ることがあります。

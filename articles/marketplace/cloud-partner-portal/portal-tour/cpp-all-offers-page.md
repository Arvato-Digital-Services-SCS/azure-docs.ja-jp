# <a name="all-offers-page"></a>[すべてのプラン] ページ

[ **[すべてのプラン]** ](https://cloudpartner.azure.com/#alloffers) ページでは、パートナーは作成したオファーの一覧と、各オファーの名前、種類、公開元、現在の状態を表示できます。  結果をフィルター処理するための文字列を指定できます。   ページ下部の状態領域には集計された件数とページごとの件数が表示され、一覧ページ間を移動できます。 この一覧にあるオファーを選択すると、関連付けられているオファー エディターにそのオファーが表示されます。

## <a name="next-steps"></a>次の手順

次のナビゲーション メニュー バー項目は、承認されたオファーが一覧表示される [[承認] ページ](./cpp-approvals-page.md) に表示されます。  承認されたオファーは既に公開済み、または公開途中のいずれかです。

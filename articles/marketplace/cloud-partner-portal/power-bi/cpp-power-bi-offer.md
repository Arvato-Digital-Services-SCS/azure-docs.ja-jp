# <a name="power-bi-app-offer"></a>Power BI アプリ オファー

Cloud パートナー ポータルでは、Power BI アプリ オファーを作成、管理できなくなりました。 この機能は Microsoft [パートナー センター](https://partner.microsoft.com/pcv/)に移行されました。 詳細については、「[Microsoft AppSource 用の Power BI アプリを作成する](https://docs.microsoft.com/azure/marketplace/partner-center-portal/create-power-bi-app-offer)」を参照してください。

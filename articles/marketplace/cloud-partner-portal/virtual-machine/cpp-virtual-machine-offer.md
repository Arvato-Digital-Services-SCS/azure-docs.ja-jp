# <a name="azure-virtual-machine-offer"></a>Azure Virtual Machine プラン

[Cloud パートナー ポータル](https://cloudpartner.azure.com/)では、Azure Virtual Machine プランの作成と管理がサポートされなくなりました。 この機能は Microsoft [パートナー センター](https://partner.microsoft.com/pcv/)に移行されました。 詳細については、[Azure Virtual Machine プランの作成](../../partner-center-portal/azure-vm-create-offer.md)に関するページをご覧ください。

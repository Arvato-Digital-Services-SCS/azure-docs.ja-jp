# <a name="marketing-best-practices"></a>マーケティングのベスト プラクティス

最適化されたマーケットプレースのリストや Go-To-Market キャンペーンは、顧客獲得の増加を促進する助けになるため、クラウド ビジネスの戦略や原動力の点で非常に重要役割を果たすことができます。 オファーの一覧を示し、顧客試用版を提供し、Microsoft のお客様やパートナー コミュニティとつながる準備を整えてください。 [AppSource と Azure Marketplace のマーケティング ベスト プラクティス ガイド](https://aka.ms/marketplacebestpractices)をダウンロードして、オンラインのマーケティング活動から最大限の成果を引き出す方法を学んでください。

**次の手順**

マーケットプレース GTM サービスの詳細については、「[Go-to-Market サービス](https://partner.microsoft.com/reach-customers/gtm)」を参照してください。

[パートナー センター](https://partner.microsoft.com/dashboard/account/v3/enrollment/introduction/partnership)にサインインして、オファーを作成して構成します。

# <a name="streaming-catalog-views-transact-sql"></a>ストリーミング カタログ ビュー (Transact-SQL)

このセクションでは、T-SQL ストリーミングに関連する使用可能なカタログ ビューと関数について説明します。
  
## <a name="in-this-section"></a>このセクションの内容  
  
|表示|説明|  
|:---|:---|
|[sys.external_streams](sys-external-streams.md) |データベースのスコープ内で作成された外部ストリーム オブジェクトごとに 1 行のデータを返します。|
|[sys.external_streaming_jobs](sys-external-streaming-jobs.md) |データベースのスコープ内で作成された外部ストリーミング ジョブごとに 1 行のデータを返します。|
|[sys.external_job_streams](sys-external-job-streams.md)|外部ストリーミング ジョブにマップされた入出力外部ストリーム オブジェクトごとに 1 行を返します。|

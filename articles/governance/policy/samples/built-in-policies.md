# <a name="azure-policy-built-in-policy-definitions"></a>Azure Policy の組み込みのポリシー定義

このページは、Azure Policy の組み込みのポリシー定義のインデックスです。

各組み込みの名前は、[Azure Policy GitHub リポジトリ](https://github.com/Azure/azure-policy)のポリシー定義ソースにリンクしています。 組み込みは、**メタデータ**内の**カテゴリ** プロパティによってグループ化されます。 特定の**カテゴリ**にジャンプするには、ページの右側にあるメニューを使用してください。 それ以外の場合は、<kbd>Ctrl</kbd>-<kbd>F</kbd> を使って、ブラウザーの検索機能を使用します。

キーとエンドポイントには、LUIS ポータルの [Manage]\(管理\) セクションの [Azure Resources]\(Azure リソース\) ページからアクセスできます。 
 
1. LUIS ポータルにサインインします。 
1. アプリの一覧から自分のアプリを選択します。
1. **[Manage]\(管理\)** 、 **[Azure Resources]\(Azure リソース\)** の順に選択します。 
1. Starter リソースのキーとエンドポイントの値をコピーします。

    **Starter** キーを使うと、すべての LUIS アプリで、オーサリング API への要求は無料で、また予測エンドポイント API へのクエリは 1 か月あたり 1000 回まで行うことができます。

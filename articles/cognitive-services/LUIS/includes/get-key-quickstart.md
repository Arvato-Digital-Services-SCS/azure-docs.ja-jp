キーとエンドポイントは、LUIS ポータルの **[Manage]\(管理\)** セクションの **[Azure リソース]** ページにあります。 
 
1. [LUIS ポータル](https://preview.luis.ai/)にサインインします。 
1. 新しいアプリを作成するか、アプリの一覧から既存のアプリを選択します。
1. **[Manage]\(管理\)** 、 **[Azure Resources]\(Azure リソース\)** の順に選択します。 
1. Starter リソースのキーとエンドポイントの値をコピーします。 これらは、以下のサンプルを実行するために必要です。
   > [!NOTE]
   > **Starter** キーを使うと、すべての LUIS アプリで、オーサリング API への要求は無料で、また予測エンドポイント API へのクエリは 1 か月あたり 1000 回まで行うことができます。

## <a name="select-subscription-and-authoring-resource"></a>サブスクリプションと作成リソースを選択する

LUIS ポータルで、アプリの一覧を表示する場合、またはアプリを作成する場合は、サブスクリプションと作成リソースを選択します。 アプリには、LUIS 作成リソースからアクセスされます。

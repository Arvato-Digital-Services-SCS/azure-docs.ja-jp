# <a name="authoring-api-changes"></a>オーサリング API の変更点 

オーサリング API は V3 で更新され、[機械学習エンティティを使用したモデル分解](luis-concept-data-extraction.md)が可能になりました。 

V2 またはプレビュー V3 APIS で作成できます。 同じアプリで V2 または V3 の予測エンドポイント応答を使用することもできます。 エンドポイント応答の作成と予測に同じバージョンの API を使用する必要はありません。

LUIS ポータルでは、V3 オーサリング API のみを使用します。

## <a name="next-steps"></a>次の手順

V3 API の[予測エンドポイントの変更点](luis-migration-api-v3.md)を確認します。

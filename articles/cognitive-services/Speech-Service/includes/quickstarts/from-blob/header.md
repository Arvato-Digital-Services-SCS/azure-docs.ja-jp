このクイックスタートでは、REST API を使用して、バッチ処理でファイルから音声を認識します。 バッチ処理を使用して、ユーザーによる操作なしで音声の文字起こしを実行します。 コンカレンシー、カスタム音声モデル、またはその他の詳細を管理する必要のない、シンプルなプログラミング モデルが提供されます。 高度な制御オプションが提供される一方で、Azure 音声サービス リソースを効率的に使用できます。

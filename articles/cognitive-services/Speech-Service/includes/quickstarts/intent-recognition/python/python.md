## <a name="build-and-run-your-app"></a>アプリをビルドして実行する

コンソールまたは IDE からサンプルを実行します。

    ````
    python quickstart.py
    ````

その後 15 秒間、マイクからの音声入力が認識され、コンソール ウィンドウにログが記録されます。
